# Calistant Coming Soon

A high-performance, mobile-first coming soon page for Calistant.

## 🚀 Live Demo

Visit the live site: [https://saljug.github.io/calistantcomingsoon/](https://saljug.github.io/calistantcomingsoon/)

## ✨ Features

- **Lightning Fast**: Optimized for performance with minimal resources
- **Mobile First**: Responsive design that works on all devices
- **Modern Design**: Clean, professional gradient background matching Figma design
- **SEO Optimized**: Complete meta tags and Open Graph support
- **Accessibility**: WCAG compliant with reduced motion support
- **Progressive Enhancement**: Works with JavaScript disabled

## 🛠️ Technologies

- Pure HTML5 & CSS3
- Modern CSS features (Grid, Flexbox, Custom Properties)
- Google Fonts (Inter) with font-display optimization
- CSS animations with accessibility considerations

## 🔧 Development

To run locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/saljug/calistantcomingsoon.git
   cd calistantcomingsoon
   ```

2. Open `index.html` in your browser or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   ```

## 🚀 Deployment

This site automatically deploys to GitHub Pages via GitHub Actions when pushed to the main branch.

### Manual Deployment

1. Enable GitHub Pages in repository settings
2. Select "GitHub Actions" as the source
3. Push to main branch - deployment will trigger automatically
